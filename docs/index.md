# Welcome to IDhub

IDhub is the [IBDGC's](https://www.ibdgc.org/) centralized platform for managing, curating, and sharing biobank and clinical identification data. It provides a consistent, reliable, and auditable system for ID integration. The database web front-end can be found at:

- [idhub.ibdgc.org](https://idhub.ibdgc.org)

The IBDGC Integrated Data Hub (IDhub) is a ID data governance system designed to:

- :material-database-check: **Centralize** biobank and clinical IDs from multiple sources.
- :material-shield-check: **Validate** metadata quality and and linkages before loading.
- :material-identifier: **Manage** global subject identifiers (GSIDs) across the consortium. ([Learn what a GSID is.](what-is-a-gsid.md))
- :material-history: **Track** complete data lineage and audit trails.
- :material-robot: **Automate** data pipelines with robust, repeatable workflows.
